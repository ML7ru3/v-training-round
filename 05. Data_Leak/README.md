# Data Leak

Ở phần này yêu cầu:
- Phân tích được cơ chế hoạt động của mã độc đánh cắp thông tin.
- Mô phỏng quá trình xử lý Data Leak/Brand Abuse thực tế.
- OSINT & Reconnaissance (Kỹ năng săn tìm): Áp dụng tư duy Hacker đẻ tìm tài sản bị lộ,
- Investigation & Digital Forensics (Truy vết sự cố): Phân tích Log để xác định hành vi lấy cắp dữ liệu (Exfiltration).

Đến checkpoint sẽ cần phải làm phân tích Log Stealer mà mentor cho trước.
