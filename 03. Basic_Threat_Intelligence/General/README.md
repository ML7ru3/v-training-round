# General

Các kiến thức nền tảng về hạ tầng mạng và bảo mật, bao gồm:

- **Internet**: Khái niệm cơ bản về Internet và cách nó hoạt động.
- **TCP/IP**: Giao thức truyền thông cốt lõi của Internet.
- **DNS**: Hệ thống phân giải tên miền.
- **HTTP(s)**: Giao thức truyền tải siêu văn bản.
- **Port & Services**: Cổng dịch vụ và các giao thức tương ứng.
- **Client - Server Model**: Mô hình máy khách - máy chủ.
- **API**: Giao diện lập trình ứng dụng.
- **Security Control**: Các giải pháp bảo mật như WAF, IDS/IPS, EDR, SIEM.
